# Carpeta Imágenes

Autor: Edgar Chen

Asesor: Dr. Luis Alberto Rivera Estrada

Se incluyen las mediciones realizadas en puntos específicos de cada circuito, los cuales están indicados en el documento titulado “Revisión de gráficas”. Además, se presentan mediciones adicionales obtenidas al variar la resistencia indicada en dicho documento.

Cada imagen se encuentra numerada al inicio con un número que indica el documento de datos .csv que le pertenece. En el nombre de cada archivo también se especifican, en orden de izquierda a derecha, el valor de la resistencia de prueba, la frecuencia y el voltaje de la señal de entrada.

Por ejemplo, a la imagen denominada 1Resistencia10o1Hz5V le corresponde el archivo de datos Imagen1.csv, el cual contiene los datos asociados a dicha medición. Este nombre indica que el experimento se realizó utilizando una resistencia de 10 Ω, con una señal de entrada de 1 Hz de frecuencia y 5 V de amplitud, generada por la fuente de alimentación.
